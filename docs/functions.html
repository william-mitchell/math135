Topic 3 Functions

3.1 Steps of the Modeling Cycle

  1. Ask a question about reality.
  2. Make some observations and collect the corresponding data.
  3. Conjecture a model or modify a known model based on the data.
  4. Test the model against known data (from step 2.) and modify the model as needed.
  5. Repeat steps 2.-4. to improve the model

3.2 Goals

  • Explain what a function is
  • Identify and explain independent and dependent variables
  • Recognize functions in a variety of forms
  • Find a functionâ€™s intercept(s)
  • Identify regions where a function is increasing/decreasing
  • Compute the average rate of change between two points on a function
  • Determine where a function is concave up/down
  • Compute the relative change between two points on a function

3.3 Tasks

  1. A family needs to purchase life insurance. What possible inputs might be used to determine the premium (monthly payment) for life insurance?

  2. In the Andes mountains in Peru, the number \(N\) of species of bats is a function of the elevation \(h\) in feet above sea level.

    1. What is the independent variable?
    2. What is the dependent variable?
    3. Interpret the statement \(f(500)=200\) if \(f\) represents this function.
    4. Would you guess this function to be increasing or decreasing?

  3. Sketch each of the following:

    1. An increasing function which is concave up
    2. An increasing function which is concave down
    3. A decreasing function which is concave up
    4. A decreasing function which is concave down

  4. What is the relative change in the Dow Jones average from 169.84 to 77.90 (from 1 January 1931 to 31 December 1931)? Compare this to the relative change in the Dow Jones average from 35,443.82 to 35369.09 (from 2 September 2021 to 3 September 2021).

  5. One rainy summer day, measurements were recorded (in inches) from a rain gauge every hour (in the table below). What was the average rate of rainfall from 8 a.m. to 5 p.m.? What are the units of this rate of change?
